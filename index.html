Server Manager roots

Nethgui framework (2011)

Nethgui

  • Simplify the development of NS modules
  • Produce a consistent look & feel
  • Be compatible with text-only browsers

Nethgui: features etc.

  • Server-side PHP programming
  • User authentication & authorization (sudo)
  • User input validation
  • Low-level API wrappers (db/config, signal-event, validate)
  • Background task progress tracking
  • Accessibility

Developer's POV

What's wrong with Nethgui

  • Lack of documentation
  • Do-it-yourself PHP framework
  • Cannot decouple UI behavior and system programming
  • Hard to implement a non-basic use case (e.g. Account provider wizard)
